# 暑期脑科学学习工具箱

面向小学 3-5 年级学生的暑期认知训练工具集，基于工作记忆、抑制控制、执行功能等脑科学研究设计。

## 快速开始

1. 用浏览器打开 **`index.html`**（工具箱首页）
2. 按「今日推荐训练顺序」依次完成，或从工具列表自由选择
3. 每天总量约 30 分钟：早上认知训练 + 晚上学科检查

**在线访问地址：** https://liushuying-kathy.github.io/sumtool/

## 文件结构

```
暑期学习工具卡/
├── index.html                    ← 入口页（从这里开始）
├── 暑假训练规划系统.html          ← 60-90天完整方案
├── 数字倒背与变式填空卡.html      ← 工作记忆训练
├── 斯特鲁普进阶练习卡.html        ← 抑制控制训练
├── stroop-cards/                  ← 10张打印训练卡（PNG）
│   ├── stroop_test_card_1.png
│   └── ... (共10张)
├── 多步骤家务指令卡.html          ← 执行功能训练
├── 口述解题法话术卡.html          ← 口述解题三步法
├── 学科检查卡.html               ← 语数英检查清单
├── 错误模式归因卡.html            ← 错题归类与改进
├── 双周压力测试方案卡.html        ← 双周综合测验
└── 奥本海默直觉训练阶段明细.html   ← 五阶段理论参考
```

> **注意**：斯特鲁普训练卡片位于 `stroop-cards/` 文件夹（10 张 PNG），HTML 已自动加载。

## 部署到 GitHub Pages

1. 进入仓库 **Settings → Pages**
2. Source 选择 **Deploy from a branch**
3. Branch 选 `main`，文件夹选 `/ (root)`
4. 保存后等待 1-2 分钟，访问：https://liushuying-kathy.github.io/sumtool/

## 发给小朋友使用

- 分享 GitHub Pages 链接，或把整个文件夹打包发给孩子/家长
- 建议固定一台设备（平板/电脑），数据保存在浏览器本地
- 第 1-2 周家长陪同，第 5 周起可完全自主

## 数据说明

- 训练数据保存在浏览器 `localStorage`，换设备或清缓存会丢失
- 涉及积分、错误记录、双周测试分数等会自动保存
- 无需联网，离线可用

## 每日训练节奏

| 时段 | 工具 | 时长 |
|------|------|------|
| ☀️ 早上 | 数字倒背 → 斯特鲁普 → 家务挑战 | ~20分钟 |
| 🌙 晚上 | 口述解题 → 学科检查 → 错误归因 | ~10分钟 |
| 📊 双周六 | 双周压力测试 | ~30分钟 |
